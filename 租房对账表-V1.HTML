分类 项目 检查要点与判断标准
周边与配套 周边环境 楼层与电梯：所在楼层（高/低层优缺点）；是否有电梯、电梯数量与高峰期等待时间；停电时楼梯情况；顶层注意漏水、底层注意潮湿 / 蚊虫。
噪音测试：窗外交通噪音（白天 + 夜间各去一次）；楼上楼下脚步声；电梯井噪音；邻里素质（夜间安静度）。
周边配套：垃圾转运站 / 垃圾房位置（有无异味）；餐饮店（油烟 / 气味 / 蟑螂）；夜市 / 烧烤摊 / 大排档；工厂 / 变电站 / 加油站 / 高压线；学校 / 医院噪音源。
地铁距离与步行时长：最近地铁站名称、步行 / 骑行 / 公交实际时长；早晚高峰实测通勤时间；末班车时间（决定夜归是否方便）。
公共区域 公共安全：监控覆盖（楼道 / 电梯 / 出入口 / 停车场）；门禁系统（刷卡 / 密码 / 门铃 / 对讲）；楼道灯与楼道清洁；快递柜位置（户外 vs 户内）；外卖存放区；停车场 / 单车棚。
网络信号：手机 4G / 5G 实测（每个房间都要试）；Wi-Fi 覆盖（每个房间）；宽带入户（是否已装、运营商、费用）；信号死角（电梯 / 卫生间 / 阳台）。
晾衣区：是否有阳台（独立阳台 vs 公共阳台）；阳台是否封闭（开放易被雨淋 / 丢衣）；是否有晾衣杆 / 晾衣绳；衣服会不会被偷；公共晾衣区位置；朝向（南向晾衣最快）。
阳台：是否有阳台；朝向（南向最优，采光 + 晾衣）；是否封闭（开放 vs 封闭式）；用途（晾衣 / 储物 / 休闲 / 种植）；防水情况（地漏 / 排水）；护栏高度（安全性）。
硬装 墙面 整体墙面：有无明显裂缝、鼓包、发霉、返潮、水渍；用手背轻擦看是否掉灰掉粉。
墙纸 / 墙布：有无翘边、起泡、污渍、破损、霉斑。
瓷砖（厨卫）：有无空鼓（敲击听声）、脱落、勾缝发霉、渗水痕迹。
平整度：用手电侧照，看有无明显凹凸；阳角阴角是否垂直。
护墙板 / 木饰面：有无变形、起翘、脱钉、发霉。
地面 木地板：有无翘起、划痕、霉变、踩踏异响、空鼓、色差。
地砖：有无空鼓、开裂、高低差、崩边。
门槛石 / 过门石：有无断裂、与两侧地面缝隙大小、密封胶状态。
踢脚线：有无脱落、发霉、缺口、缝隙过大。
天花板 顶面涂料：有无水印、发黄、开裂、霉斑；重点查楼上邻居漏水迹象。
吊顶：有无下垂、变形、裂缝、塌陷。
水电 总闸 / 漏保：能否正常合闸、有无跳闸、漏保测试按钮是否有效。
插座数量与位置：床头、沙发旁、书桌、卫生间（带防溅盖）、厨房是否充足。
插座通电：每个插座都用充电器 / 电笔实测，看是否带电、是否虚接。
开关控制：是否一一对应、有无失灵、双控是否正常。
水龙头出水：水压是否正常、冷热水是否正常切换、是否有滴漏。
地漏排水：放满一桶水看下水速度、是否反味（重点查卫生间和阳台）。
防水：墙角、卫生间相邻墙面、楼下邻居天花板有无渗水痕迹（关键项）。
门窗 入户门：锁芯能否正常开合、有无异响、防盗链 / 猫眼是否齐全。
室内门：开关是否顺畅、有无刮蹭地面、门锁是否正常、门吸是否齐全。
窗户：能否正常开合、密封条是否老化、有无漏风、晃动。
纱窗：是否齐全、能否正常使用、网面有无破损。
玻璃：有无裂纹、密封胶是否完好、是否有雾气（双层玻璃漏气）。
定制柜 / 固定家具 橱柜：柜门开合顺畅度、拉篮抽屉、五金件（铰链 / 阻尼）是否正常。
衣柜 / 衣帽间：移门 / 平开门、隔板、五金、有无虫蛀发霉。
玄关柜 / 鞋柜 / 飘窗柜：同上检查。
厨卫固定设施 马桶：冲水力度、是否漏水、釉面有无裂、冲水按钮是否回弹。
洗手台：下水是否顺畅、台面有无裂、龙头是否正常。
浴缸 / 淋浴房：下水速度、玻璃胶是否发霉、挡水条是否严密。
花洒：出水量、能否正常切换模式、有无滴漏、软管有无破损。
燃气灶：能否正常点火、火苗颜色（蓝色正常）、熄火保护是否有效。
油烟机：吸力、噪音、滤油盒情况、能否正常运行。
热水器：能否正常点火 / 启动、加热速度、水温是否稳定、有无漏气。
排气扇 / 风暖：能否正常运行、噪音大小、卫生间是否有反味。
软装 家具 床：尺寸（1.5m / 1.8m）、有无异响、床垫软硬度、有无霉味、床头靠背完整。
沙发：有无破损、塌陷、污渍、异味、坐垫回弹。
餐桌椅：稳固性、桌面有无划痕、椅子有无摇晃。
衣柜（独立式）：尺寸、移门 / 平开门、隔板是否完整、有无虫蛀。
书桌 / 工作台：尺寸、稳固性、是否够放显示器 / 笔记本。
椅子（办公椅 / 餐椅）：升降、滚轮、扶手是否正常。
家电 空调：制冷 / 制热效果、噪音大小、滤网是否干净；重点：当前档位是否最耗电。
冰箱：制冷效果、运行噪音、结霜情况、容量、冷冻室是否正常。
洗衣机：能否正常运行、噪音、脱水效果、洗涤模式选择。
电视：能否正常开机、屏幕有无亮点 / 暗点、遥控器是否齐全。
微波炉 / 烤箱：能否正常加热、门封是否严密。
电热水器 / 燃气热水器：加热速度、水温稳定性、有无漏水漏气。
小家电：电饭煲、热水壶、吹风机等是否配备（看合同清单）。
智能门锁：指纹 / 密码 / 钥匙是否都能用、电量提示。
布艺 / 织物 窗帘：是否齐全、能否正常拉合、有无发霉、遮光性。
地毯：有无污渍、发霉、掉毛。
床品 / 沙发套：是否配备、是否干净、有无异味。
可移动灯具 落地灯 / 台灯：是否配备、灯泡是否亮、开关是否正常。
装饰灯（灯带 / 小夜灯）：是否配备。
灯泡备用：是否有备用、型号是否统一。
装饰 / 其他 挂画 / 装饰品：是否配备、有无损坏（一般不重要）。
绿植：是否保留（合同里通常不算）。
灰区 空调类型 壁挂 / 柜机 → 软装，可拆走；确认品牌型号、运行年限。
中央 / 风管机 → 硬装，走管线和吊顶；确认使用情况和维修责任。
嵌入式家电 嵌入橱柜的烤箱 / 洗碗机 / 蒸箱 → 硬装；确认能否正常使用、是否预留电源。
独立摆放的微波炉 / 电饭煲 → 软装。
灯具归属 吸顶灯 / 筒灯 / 灯带 / 吊灯（接线款） → 硬装，线路接死在天花板。
落地灯 / 台灯 / 插电款装饰灯 → 软装。
窗帘归属 通常归软装可拆走，但部分房东要求保留；合同需明确。
合同 家具家电清单附件 是否详细列出品牌、型号、数量、新旧程度；作为退租验收依据。
押金扣除规则 硬装 vs 软装损坏的扣款标准是否明确；自然损耗 vs 人为损坏的界限。
维修责任划分 家电、家具、水电故障由谁修；响应时限；费用承担方式。
退租验收标准 是否要求恢复原状；是否拍照存档；是否当场验收签字。
转租 / 续约 能否转租；续约优先权；续约时租金调整规则。
中介费 / 服务费 谁付（房东 / 租客 / 各付一半）、何时付、付多少。
付款方式 押一付三 / 押二付一 / 月付；付款日；逾期违约金。
水电燃气网络 过户流程；缴费方式；表底数确认；欠费谁承担。
提前退租 违约金比例；是否允许转租；押金退还条件。
房屋用途限制 是否允许做饭 / 养宠物 / 接待访客 / 注册公司 / 群租。
