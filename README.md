# 租房对账表

一份在租房阶段使用的检查清单，帮助你系统性看房、避免遗漏关键问题。

在线地址：https://thebear617.github.io/rental-checklist/

## 什么时候用

确定要租房、准备联系中介看房时开始使用。覆盖从第一次看房到签合同的完整流程，每个阶段逐步推进，确保不遗漏任何细节。

## 三个阶段

- **第一次看房**：线上问清费用和基本信息，实地检查周边环境、房屋配置、安全问题。排除有硬伤的房源。
- **第二次看房**：带着第一次发现的问题重点复查。逐项深度检查墙面、地面、插座、家电，全屋拍照留底。
- **合同注意事项**：签合同前逐条确认条款，拍照记录水电气表读数和已有瑕疵，保障押金能全额退还。

## 使用方式

1. 打开网页，按当前阶段选择对应 Tab
2. 按卡片顺序逐项检查，每个问题都附有提示说明
3. 用搜索框快速定位关心的项目（如「插座」「空调」「漏水」）
4. 可以展开/收起所有卡片，方便逐项核对

## 说明

- 本清单仅供参考，实际以房屋现状和租赁合同约定为准
- 检查项基于真实租房经验整理，欢迎按自身需求调整
